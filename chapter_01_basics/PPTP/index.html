02 - JS Step By Step

02_JS_Step_By_Step.js Variables & Functions

This example covers declaring variables with let, using for loops to iterate, and defining functions. It demonstrates how JavaScript executes code step by step, including calling a function inside a loop.

let a = 10;
console.log(a);

for (let a = 0; a < 100000; a++) {
    console.log(a);
    print();
}

function print() {
    console.log("Hello");
}

03 - Verify Setup

03_verify_setup.js Environment

This script verifies your Node.js setup by printing the platform (Darwin for Mac, Win32 for Windows, Linux for Linux), architecture (x64, arm64), and Node.js version. Essential for confirming your development environment is ready.

console.log(process.platform);
// MAC - DARWIN
// WINDOWS - WIN32
// LINUX - LINUX

console.log(process.arch);
// x64
// arm64


console.log("Node Version:", process.version);

04 - Hot Code

04_hot_code.js Performance

Demonstrates a simple function call performance test. The add function is called 10,000 times in a loop, illustrating how JavaScript handles repetitive function invocations. This concept is relevant to understanding V8 engine optimizations (hot code paths).

console.log("Hello");

function add(a, b) {
    return a + b;
}
let result;
for (let i = 0; i < 10000; i++) {
    result = add(i, i + 1);
}
console.log("After 10000 calls:", result);
